For the installation and use of the Spanish version of Text2Onto, the user needs to have installed the following software:

-TreeTagger found through http://www.ims.uni-stuttgart.de/projekte/corplex/TreeTagger/ . This application requires the installation of a Perl interpreter, and the Spanish parameter file must be placed in the parameters directory.

-GATE 4.0, found through http://gate.ac.uk/ 

-WordNet 2.0, found through http://wordnet.princeton.edu/oldversions 

-For the correct performance of the system using the Spanish WordNet Algorithm, the user should apply for a license of use of the Spanish WordNet files. This license can be obtained from http://www.lsi.upc.edu/~nlp/web/index.php?option=com_content&task=view&id=31&Itemid=57 

-Then, the user should ask through JOHANNA, PUT HERE THE MAIL for the Spanish WordNet files modified to be utilized in Text2Onto

All the applications and files above mentioned must be installed in directories without empty spaces for the correct performance of the system.

When all the software requirements are satisfied, the user has to download the installation pack from JOHANNA, PUT HERE THE LINK and execute myInstallationSpanish.jar. The installation is self-explanatory, and the user only have to fill the paths of the applications previously installed.

The last step will be the modification through any editor of the “text2onto.properties” file replacing all back slashes “\” by forward slashes “/”

Finally, execute Text2Onto.bat
